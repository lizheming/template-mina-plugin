MiniProgram Plugin Scaffold template for Tina.js

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao mina-plugin my-project
```

### From git

```bash
sao lizheming/template-mina-plugin my-project
```
